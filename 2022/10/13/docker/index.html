docker的常用命令

帮助命令

docker version #显示docker的版本信息

docker info #显示docker的系统信息，包括镜像和容器的数量

docker 命令 --help #万能命令

帮助文档地址：https://docs.docker.com/reference/

镜像命令

  1. docker images #查看本地所有主机上的镜像

-a, --all #显示所有镜像

-q, --quiet #只显示镜像id

  1. docker search 搜索镜像
sudo docker search mysql
NAME                            DESCRIPTION                                     STARS     OFFICIAL   AUTOMATED
mysql                           MySQL is a widely used, open-source relation…   13234     [OK]
mariadb                         MariaDB Server is a high performing open sou…   5062      [OK]

percona                         Percona Server is a fork of the MySQL relati…   588       [OK]
bitnami/mysql                   Bitnami MySQL Docker Image                      77                   [OK]

–filter=STARS=3000 #搜索出来的镜像就是STARS大于3000的

  1. docker pull 下载镜像

#下载镜像 docker pull 镜像名[:tag]（tag是对应版本）

docker pull mysql #如果不写tag，默认是latest（最新版）

  1. docker rmi #删除镜像

docker rmi -f 镜像id

docker rmi -f $(docker images -aq) #利用linux命令，此时会将所有镜像递归删除

容器命令

说明：我们有了镜像才可以创建容器。linux，下载一个centos镜像来测试学习

docker pull centos #安装最新版的centos

新建容器并启动

docker run [可选参数] image

#参数说明
--name="Name"   #容器名字 tomcat01 tomcat02  用来区分容器
-d    #后台方式运行
-it   #使用交互方式运行，进入容器查看内容
-p    #指定容器的端口 -p 8080：8080
	-p ip:主机端口:容器端口
	-p 主机端口:容器端口
	-p 容器端口
-P    #随机指定端口

docker run -it centos /bin/bash #启动并进入容器

exit #从容器中退回去主机中

  1. docker ps #列出所有运行中的容器

-a #查看运行中的容器+运行过的容器

-q #只显示容器的id

  1. docker rm 容器id #删除指定的容器

docker rm -f $(docker ps -aq) #删除指定的容器

docker ps -a -q|xargs docker rm #删除所有的容器

  1. docker 启动暂停容器

docker start 容器id #开始容器

docker stop 容器id #停止容器

docker restart 容器id #重启容器

docker kill 容器id #暴力停止容器
